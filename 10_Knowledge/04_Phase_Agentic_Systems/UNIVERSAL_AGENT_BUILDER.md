## 🚀 TOOL COMMANDS REFERENCE

### Pi (Gemini CLI) - Your Main Tool
```bash
# Interactive
pi

# One-shot
pi -p "your prompt"

# With MCP server
pi -e /path/to/server.py "prompt"

# Continue session
pi --continue
```

### Goose
```bash
# One-shot
goose run -t "prompt"

# Interactive
goose session

# With MCP
goose run -t "prompt" --with-extension "python server.py"
```

### Fallback Wrapper
```bash
# Auto-switches to Gemini if Ollama fails
pifallback "prompt"
```
