# Personal Knowledge Management & Learning Roadmap

## History
This vault originated on **Linux Mint** and migrated through several distributions before settling on **Fedora** on **March 13, 2026**. This migration marked a shift towards a more structured, autonomous learning infrastructure.

## Core Strategy
This repository serves as a **Personal Knowledge Management (PKM)** system and a **Long-term Learning Roadmap**. It tracks progress from foundational computational concepts to autonomous systems.

## Key Components
- **[[Topics|Learning Roadmap]]**: The master index of the 12 learning phases. Use this to track progress and identify the next focus areas.
- **[[GEMINI.md|Agent Instructions]]**: The foundational mandates for AI assistants (like Gemini CLI) interacting with this vault.
- **[[00_daily_logs/|Daily Logs]]**: A persistent record of daily tasks and accomplishments.

## Organization
The vault is structured with numerical prefixes to ensure a logical flow:
- `00_daily_logs/`: Runtime logs.
- `10_Knowledge/`: Categorized knowledge base across 12 phases.
- `99_System/`: Meta-docs, configurations, and agent-specific notes.
