# Syllabus Builder Skill
**Purpose:** Construct and update the hierarchical learning tree of the "Full Learning Curve."
**When to use:** When completing topics, progressing through phases, or adding new learning chapters.
**Steps:**
1. Scan `10_Knowledge/` to find all current learning notes.
2. Open `SYLLABUS.md` at the vault root.
3. Organize notes by Phase -> Project -> Chapter -> Topic.
4. Add a one-line summary for each chapter.
5. Update the status: `[x]` for completed, `[/]` for in-progress, `[ ]` for pending.
6. Verify that every topic note has a [[Phase header block]].
**Output:** A hierarchical, linked map of all learning in `SYLLABUS.md`.
