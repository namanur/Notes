# Study Builder Skill

This skill enforces a brutal, execution-first approach to learning.

## Instructions
1. **Identify Topic:** Reference `ROADMAP.md` for the current context.
2. **Apply Constraints:**
   - **Mental Model:** DAG, snapshots, pointers. No fluff.
   - **Modern Only:** `git switch`, `git restore`. No `git checkout` for branching.
   - **Mastery Drills:** Create, break, fix.
   - **Execution Contract:** Timebox, Start Command, Completion Condition.
3. **Automate Integration:**
   - Add [[Phase X]] header.
   - Link `ROADMAP.md`.
   - Update `99_System/Topics.md`.
